# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1fjnj3a8m2fawDd3lsxnKe2dEgjhp-SH9

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
## Example interactions with the APP

1. Find route from SFO to FishermansWharf with a stop at nearby cafe and show me nearby cafes.
2. From AT&T stadium to EPIC waters indoor waterparks. With pitstops at important landmarks.
